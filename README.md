# This is a script used for educational and local use that crasks passwords by bruteforce! (everything is run in local terminal)

  
**USAGE**:  
`CombinationsCalc` -  
folder contains two sub folders.  
`Version1` only work with numbers.  
`Version2` is working with every possible character combination.
Start the program and from there everything will be self explanatory.  

`SHAhashGenerator` -  
is used to encript your password. Run the program, type in password you want encrypted and it will put out your password in 3 different encription formats.

`PasswordCracker` -  
folder has 3 versions of `cracker.py` files. Each one with more complexity added to it.  
`Version1` and `Version2` only work with numbers.  
`Version3` is working with every character combination.
When you run the program everything should be self explanatory.   
You put your SHAhash(encripted password) and type of password it is. Then it is going to brute force it untill its cracked.
